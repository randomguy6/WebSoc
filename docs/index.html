Project Description

My project is Web Communication between Server and Client using Web Socket technology, which provides services more quickly than traditional HTTP connection, that is designed for data centers to communicate any events from Servers to all registered Client. My research will include basic networking between server and client, how to build a web page from HTML and JavaScript, and learning new programming languages such as JavaScript and HTML. The technology I will be using is J2EE IDE for JavaScript and notepad for HTML to build any necessary Websocket code for this project. For my final presentation, I would have a web page that has the Websocket embedded to demo for the public to interact with.

Project Objectives

  • Become proficient with self teaching programming languages
  • Understand the communication between client and server using the websocket
  • Make the program useful for data centers
  • Allow communication and registration between servers and clients more easily and more efficient
  • Be familiar with how scripting languages work and how it is different than object oriented programming languages
